FFStream

版本：1.0.0

项目简介

FFStream 是一个基于 Node.js 和 FFmpeg 的流媒体管理系统，提供了便捷的 Web 界面来管理直播流。

主要功能

  • 导入新的直播流
  • 编辑现有流配置
  • 启动/停止流转发
  • 删除流配置
  • 流状态监控

安装说明

  1. 确保已安装 Node.js (>= 14.x) 和 FFmpeg
  2. 克隆项目代码
  3. 运行 npm install 安装依赖
  4. 配置 .env 文件
  5. 运行 npm start 启动服务

配置说明

主要配置项在 config/default.js 中：

  • 推流服务器：rtmp://ali.push.yximgs.com/live/
  • 拉流服务器：http://ali.hlspull.yximgs.com/live/
